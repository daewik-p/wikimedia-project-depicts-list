# Depicts List Generator (Script)

A utility script to generate a list of media files from a Wikimedia Commons category based on their structured data statements.

## Given Information

| Purpose | Example URL / Endpoint |
| :--- | :--- |
| **SPARQL Query** | `http://w.wiki/GVh5` |
| **Wikidata Entity Data**| `https://commons.wikimedia.org/wiki/Special:EntityData/M1234567.json`|

## Our Aim
For a given wikimedia commons category run a SPARQL query to list all media files in that category. Fetch the data for each image and compile them into a readable form. (Image to the left and related data to the right)
